CHAPTER 1

Guide to AutoCAD Basic

AutoCAD is a computer-aided design (CAD) software used by architects, engineers, and designers to create digital designs and drawings. Here is a brief guide to AutoCAD basics

  • User Interface: When you first launch AutoCAD, you will be presented with the main user interface. The user interface is composed of various tabs, panels, and commands that are used to create and modify drawings. You can customize the user interface by adding or removing panels, creating new tool palettes, or adjusting the size and layout of the workspace.
  • Drawing Area: The drawing area is where you create and modify your designs. You can use various tools and commands to draw lines, shapes, and other objects on the drawing area. The drawing area can be customized by setting the units, grid spacing, and other parameters.
  • Command Line: The command line is located at the bottom of the user interface and is used to enter commands and parameters. You can use the command line to access various tools and commands, set parameters, and customize the drawing area.
  • Navigation: You can navigate the drawing area using various commands such as Pan, Zoom, and Orbit. These commands allow you to move around the drawing area and zoom in or out to see more or less detail
  • Drawing Tools: AutoCAD offers a wide range of drawing tools such as Line, Circle, Rectangle, and Arc. These tools allow you to create basic shapes and lines that form the basis of your design.
  • Modifying Tools: AutoCAD also offers various modifying tools that allow you to edit and refine your design. Some of the common modifying tools include Move, Copy, Rotate, Scale, and Trim
  • Layers: Layers are used to organize your design into different categories. For example, you might create a layer for dimensions, another for text, and another for the main drawing. You can control the visibility, color, and other properties of each layer to make it easier to work with.
  • Dimensioning: Dimensioning is an essential part of any design. AutoCAD offers various tools that allow you to create and edit dimensions. You can add dimensions to your drawing, adjust their properties, and modify their appearance.
  • Printing and Plotting: Once your design is complete, you can print or plot it to a physical medium such as paper or PDF. AutoCAD offers various tools and commands that allow you to set up the printing or plotting process and customize the output.
This is just a brief guide to AutoCAD basics. AutoCAD is a powerful software with many features and tools. To become proficient in AutoCAD, you will need to practice and experiment with the different commands and tools. Additionally, there are many resources available online, including tutorials, forums, and blogs, that can help you learn more about AutoCAD and its capabilities.

The Command Window

The Command Window is a key component of the AutoCAD interface that allows you to enter and execute commands. It is located at the bottom of the AutoCAD window and provides a way to interact with the software using text-based commands. Here are some important features of the Command Window in AutoCAD

  • Prompt: The Command Window displays a prompt at the beginning of each command line. This prompt tells you what the software is expecting you to do next. For example, when you start the Line command, the prompt might say "Specify first point".
  • AutoComplete: As you type a command or parameter, AutoCAD's AutoComplete feature suggests possible matches. This can help you quickly enter commands and parameters without having to type out the entire command
  • Command History: The Command Window keeps a history of the commands that you have entered. You can use the up and down arrows on your keyboard to scroll through the command history and reuse previous commands
  • Dynamic Input: Dynamic Input is a feature that displays a tooltip near your cursor as you move it over the drawing area. The tooltip shows information about the command you are using and allows you to enter parameters without having to go back to the Command Window
  • Command Aliases: AutoCAD allows you to define command aliases, which are shortcuts for frequently used commands. For example, you can define "L" as an alias for the Line command. This can help you work more efficiently and save time
  • Command Options: Some commands in AutoCAD have multiple options or parameters that can be set. The Command Window displays these options when they are available, allowing you to set them as needed
Overall, the Command Window is an essential part of working with AutoCAD. It provides a flexible and efficient way to interact with the software, and can help you work more efficiently and effectively

The Mouse

The mouse is a fundamental tool in AutoCAD that is used to navigate the drawing area, select objects, and enter commands. Here are some ways in which the mouse is used in AutoCAD

  • Selection: To select an object in AutoCAD, you can use the mouse to click on it. You can also drag a selection window around multiple objects to select them all at once. To deselect an object, you can click on it again or press the Esc key
  • Navigation: The mouse is used to navigate the drawing area in AutoCAD. You can use the scroll wheel to zoom in and out, or press the Shift key while moving the scroll wheel to pan around the drawing. You can also use the middle button or wheel click to enter the Orbit command, which allows you to rotate the view around a specific point
  • Command Entry: In AutoCAD, you can enter commands using the mouse. For example, you can use the mouse to click on a toolbar button, which will activate a specific command. You can also right-click anywhere in the drawing area to open a shortcut menu, which provides quick access to common commands.
  • Drag-and-Drop: In AutoCAD, you can use the mouse to drag and drop objects. For example, you can select an object and then drag it to a new location. You can also use the copy command to make a copy of an object and then drag the copy to a new location
  • Object Manipulation: The mouse is also used to manipulate objects in AutoCAD. For example, you can use the mouse to move an object, stretch it, rotate it, or scale it
Overall, the mouse is a versatile tool in AutoCAD that allows you to navigate the drawing area, select objects, and enter commands. Becoming familiar with the various ways in which the mouse is used in AutoCAD can help you work more efficiently and effectively

New Drawings

To make a new drawing in AutoCAD, you can follow these steps:

  • Open AutoCAD: Open the AutoCAD software by double-clicking on the icon on your desktop or by selecting it from the start menu
  • Create a new drawing: Once the AutoCAD application is launched, you will see a start page. Click on "New Drawing" to create a new file. Alternatively, you can use the "CTRL+N" shortcut to create a new drawing
  • Choose a template: AutoCAD provides a variety of templates for different types of drawings, such as architectural, mechanical, and electrical. You can select a template that suits your needs from the list of available options. If you don't want to use a template, you can choose the "Blank" option
  • Set up the drawing: Once you have selected a template or chosen to start with a blank drawing, you can set up the drawing by specifying the units of measurement, drawing scale, and other settings. You can use the "Units" command to set the drawing units, and the "Limits" command to set the drawing area
  • Start drawing: Once the drawing is set up, you can start creating your design by using the various drawing tools in AutoCAD. You can use commands like "Line", "Circle", "Rectangle", and "Arc" to draw basic shapes. You can also use more advanced tools to create more complex designs
  • Save the drawing: As you work on the drawing, it's a good idea to save your progress periodically. To save the drawing, you can use the "Save" or "Save As" commands. You can choose a location to save the drawing and give it a name
That's it! With these simple steps, you can create a new drawing in AutoCAD and start working on your design

Create Your Own Drawing Template File

Creating your own drawing template file in AutoCAD can save you time and effort by providing a starting point for each new drawing that you create. Here are the steps to create your own drawing template file

  • Create a new drawing: Open AutoCAD and create a new drawing using the steps described in the previous answer. Choose a template that is closest to your preferred settings
  • Customize the drawing settings: Set up the drawing with the settings you want to use for all future drawings. This may include the units of measurement, drawing scale, and other settings.
  • Customize the layers: Create a set of layers that you use regularly and name them appropriately, set the color, line weight, and linetype to your preference
  • Add any commonly used objects: Add any commonly used objects to the template, such as title blocks, borders, and company logos
  • Save the drawing as a template: Once you have set up the drawing with the settings you want to use, you can save it as a template file. Go to the "Save As" command and select "Drawing Template" from the "Save as type" drop-down menu. Name the file and choose a location to save it. Make sure to save it in the "Templates" folder in the AutoCAD installation directory so that it is easily accessible.
  • Use the new template file: To use the template file in a new drawing, go to the "New Drawing" command, select "Use a Template", and choose your new template from the list.
With these steps, you can create a custom drawing template file in AutoCAD that includes your preferred settings, layers, and objects. This will save you time and effort in the long run by providing a consistent starting point for each new drawing.

Units

Units in AutoCAD refer to the measurement system that you use for your drawing. AutoCAD provides a range of units options, including metric, imperial, and other specialized systems. Here's how to set the units in AutoCAD

  • Open a new or existing drawing in AutoCAD
  • Type "UNITS" in the command line and press Enter. The "Drawing Units" dialog box will appear
  • In the "Drawing Units" dialog box, you can choose the "Type" of unit system you want to use (such as Decimal, Architectural, Engineering, or others).
  • Then, you can choose the "Precision" or the number of decimal places you want to use in your drawing
  • You can also set the "Insertion Scale" and "Scale Objects" options as desired
  • Once you have selected the desired units and precision, click "OK" to save the settings
  • You can also adjust the units for a specific drawing by typing "UNITS" in the command line and modifying the settings as described above
    • It is important to ensure that you are using the correct units for your drawing to avoid measurement errors and other issues. Once you have set the units in AutoCAD, you can start drawing with the correct measurements and dimensions.

Model Scale

Model scale in AutoCAD refers to the size and proportion of the objects and elements that you draw in your model space. The model scale determines the relationship between the size of objects in your drawing and their real-world size. Here's how to set the model scale in AutoCAD

  • Open a new or existing drawing in AutoCAD
  • Type "SCALE" in the command line and press Enter. The "Scale" dialog box will appear
  • In the "Scale" dialog box, you can set the scale factor that you want to use for your drawing. The scale factor represents the ratio between the size of the objects in your drawing and their real-world size. For example, if you set a scale factor of 1:50, each unit in your drawing represents 50 units in real life
  • You can also choose whether to apply the scale to the entire drawing or to a specific selection of objects
  • Once you have set the desired scale, click "OK" to save the settings.
By setting the model scale in AutoCAD, you can ensure that your drawing accurately represents the real-world size and proportions of the objects and elements that you are drawing. This is important for tasks such as architectural and engineering design, where accurate scaling is essential

Printing

Printing in AutoCAD involves specifying the printer, paper size, and other printing parameters to create a printed version of your drawing. Here are the basic steps for printing a drawing in AutoCAD:

  • Open the drawing you want to print.
  • Type "PLOT" or "PLOTTERMANAGER" in the command line and press Enter. The Plot dialog box or Plotter Manager dialog box will appear.
  • In the Plot dialog box or Plotter Manager dialog box, select the printer you want to use from the list of available printers. You can also add a new printer if necessary.
  • Choose the paper size you want to use from the list of available sizes.
  • Set the print scale and other printing parameters as needed. You can adjust the print scale using the "Scale" option in the Plot dialog box or Plotter Manager dialog box.
  • Preview the drawing to make sure it looks correct.
  • Click the "OK" button to print the drawing
In addition to these basic steps, there are many other printing options available in AutoCAD, such as selecting a specific area of the drawing to print, printing multiple copies of the drawing, and choosing different print quality options. You can access these options from the Plot dialog box or Plotter Manager dialog box. It's also important to make sure that the printer you are using is properly configured and has the correct drivers installed to ensure a successful print job.

Create a page set up

To create a page setup in AutoCAD, follow these steps:

  • Open the drawing you want to set up for printing.
  • Type "PAGESETUP" in the command line and press Enter. The Page Setup Manager dialog box will appear
  • Click on the "New" button to create a new page setup
  • In the "New Page Setup" dialog box, enter a name for the page setup in the "Name" field.
  • Under the "Printer/Plotter" section, select the printer or plotter you want to use for printing
  • Under the "Paper Size" section, select the paper size you want to use for printing. You can choose from standard paper sizes or create a custom paper size.
  • Under the "Plot Area" section, select the area of the drawing you want to print. You can choose to print the entire drawing, a specific area of the drawing, or a window that you define
  • Under the "Plot Scale" section, select the print scale you want to use. You can choose to use a specific scale factor or fit the drawing to the selected paper size.
  • Under the "Plot Style Table" section, select the plot style table you want to use. This determines how colors and lineweights are printed.
  • Click the "OK" button to save the page setup
Once you have created a page setup, you can use it to print your drawing by selecting it from the "Page Setup" dropdown menu in the Plot dialog box or Plotter Manager dialog box. This allows you to quickly and easily set up your drawing for printing without having to manually adjust the printer, paper size, or other settings every time.

Output to a pdf file

To output a drawing as a PDF file in AutoCAD, you can use the "Plot" command and select the "PDF" option in the "Printer/Plotter" field. Here are the steps:

  • Open the drawing you want to export as a PDF
  • Type "PLOT" in the command line and press Enter to open the Plot dialog box.
  • In the Plot dialog box, select the "PDF" option in the "Printer/Plotter" field. If you don't see the PDF option, you may need to install a PDF printer driver.
  • Under the "Paper Size" section, select the paper size you want to use. You can choose from standard paper sizes or create a custom paper size.
  • Under the "Plot Area" section, select the area of the drawing you want to print. You can choose to print the entire drawing, a specific area of the drawing, or a window that you define.
  • Under the "Plot Scale" section, select the print scale you want to use. You can choose to use a specific scale factor or fit the drawing to the selected paper size.
  • Under the "Plot Style Table" section, select the plot style table you want to use. This determines how colors and lineweights are printed.
  • Under the "PDF Options" section, you can specify additional settings such as the output directory, the PDF filename, and other PDF-specific settings.
  • Click the "OK" button to save the PDF file
Once the PDF file is created, you can open it in any PDF viewer or editor to view and edit the file as needed
